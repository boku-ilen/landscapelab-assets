# LandscapeLab Assets

A collection of 3D assets and textures for landscape visualization, all licensed under Open licenses.

Assets are licensed under CC BY-SA 4.0, unless specified otherwise in an additional LICENSE file. Most assets are either self-made or from CC0 sources. Some are licensed under other CC licenses.

We use these assets in the [LandscapeLab](https://github.com/boku-ilen/landscapelab/), a landscape visualization based on [Godot](https://godotengine.org/). Therefore, import files and some other Godot-specific metadata is included. However, the asset files themselves are engine-independent.

## Contributing

Feel free to open a Pull Request to add your own assets to this collection under the following requirements:

- All assets must be licensed under an Open license (usually a form of Creative Commons).
  - Note that some tools such as SpeedTree prohibit the free redistribution of assets!
- Assets should be relevant for realistic landscape visualization.
- 3D models and textures should be optimized for real-time usage.
- Scalable assets, such as trees, should have a normalized height of 1 unit.

Some specific assets which the collection is lacking are listed in the [Issues](https://github.com/boku-ilen/landscapelab-assets/issues), but contributions are not limited to these.
